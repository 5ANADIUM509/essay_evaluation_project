# 作文评估模型对比项目

## 项目简介
本项目用于对比不同参数量（7B/14B/70B）的DeepSeek-R1-Distill系列模型在作文生成任务上的表现，通过自动化评分和 pairwise 对比，量化分析各模型的优劣。

## 核心功能
1. **模型回答生成**：调用7B/14B/70B模型生成作文回答；
2. **自动化评分**：使用DeepSeek法官模型对单模型回答打分（1-10分）；
3. **无效评分修复**：自动检测并重试生成无效的模型回答和评分；
4. ** pairwise 对比**：对比两个模型的同题回答，输出偏好概率；
5. **可视化展示**：生成偏好分布直方图、模型对比汇总图。
